突然有一天，那个人想起了许久不光顾的github。看着空荡荡的仓库，指尖划过磨砂的键盘，眼神闪烁，一个念头就此产生。

1、需要什么

  • windows 10系统
  • github和gitee账号
  • git环境
  • 合格的浏览器
  • 一双手

2、HUGO

2.1、hugo

一个制作静态网页的工具

2.2、下载

可以直接在GitHub上下载，在我创建这个文章的时候最新版如下：

hugo 0.79.1 win64版本下载

也可直接访问官方仓库hugo

2.3、安装

下载好之后，选择一个文件夹解压好。

然后将可执行文件所在的目录地址加入系统环境变量path。

目的是为了在命令行直接使用hugo。

2.3、使用过程

选一个风水比较好的文件夹，打开命令行

  • 创建一个站点

    hugo new site [站点的存放点]

  • 创建一个文章文件

    hugo new [目录]/[文章名].md

    hugo生成文件的目录默认是content，命令行的[目录]会在content文件夹下创建，可以通过配置文件修改默认值

  • 本地服务调式

    hugo server

    以默认的配置启动本地调式。运行命令后文件会启动调试，可以在http://localhost:1313/中看到自己创建的页面

  • 生成静态页面

    hugo

    以默认配置文件生成静态页面，默认在public文件夹可以看到生成的结果

hugo的使用非常简单，只要将配置文件设置妥当，后面的使用流程就仅仅是创建文件、调试文件、转静态文件、发布到远程仓库
